# KasiTech

Premium digital technology studio website and interactive demonstration portfolio.

## Develop

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` тАФ development server
- `npm run build` тАФ production build
- `npm run start` тАФ serve production build
- `npm run lint` тАФ ESLint

## Routes

- `/` тАФ Homepage
- `/work` тАФ Portfolio router & search
- `/work/all` тАФ All 12 projects
- `/work/[slug]` тАФ Case studies
- `/demo/[slug]` тАФ Live interactive demos
- `/capabilities` тАФ What we do
- `/about` ┬╖ `/company` ┬╖ `/founder`
- `/start` тАФ Start a Project
- `/lab` тАФ Kasi Lab
